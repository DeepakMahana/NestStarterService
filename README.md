## Description

Starter Pack for Microservices.

1. [Nest](https://github.com/nestjs/nest) a typescript framework with express.
2. [MongoDB](https://www.mongodb.com) a NoSQL Database.
3. [RabbitMQ](https://www.rabbitmq.com) a message broker 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
